PADROES DE DESIGN

1. ABSTRACT FACTORY (Implicito)
   Material atua como factory base
   Subclasses (Livro, Revista, DVD) sao products

2. STRATEGY
   BibliotecaService encapsula diferentes estrategias
   (buscar, emprestar, devolver)

3. TEMPLATE METHOD
   Material define a estrutura (template) em getTipo()
   Subclasses implementam a especializacao

4. SERVICE LOCATOR
   BibliotecaService localiza recursos (materiais, usuarios)

5. REPOSITORY
   BibliotecaService atua como repositorio (armazena/recupera dados)


PRINCIPIOS SOLID

1. SRP - Single Responsibility
   ┌─────────────────────────────────────────────┐
   │ Material - Responsavel por dados do material │
   │ Usuario - Responsavel por dados do usuario   │
   │ Emprestimo - Responsavel por rastrear empres│
   │ BibliotecaService - Responsavel por orquestra│
   └─────────────────────────────────────────────┘

2. OCP - Open/Closed
   ┌──────────────────────────────────────────┐
   │ Aberto para extensao (novo tipo?)        │
   │ Fechado para modificacao                 │
   │                                          │
   │ Criar Audiolivro extends Material        │
   │ BibliotecaService funciona automaticamente│
   └──────────────────────────────────────────┘

3. LSP - Liskov Substitution
   ┌──────────────────────────────────────────┐
   │ Qualquer Material pode substituir outro   │
   │ Livro, Revista e DVD sao intercambiáveis │
   │ em contextos que usam Material           │
   └──────────────────────────────────────────┘

4. ISP - Interface Segregation
   ┌──────────────────────────────────────────┐
   │ Emprestavel - para operacoes de emprést │
   │ Buscavel - para operacoes de busca       │
   │ Interfaces pequenas e focadas            │
   └──────────────────────────────────────────┘

5. DIP - Dependency Inversion
   ┌──────────────────────────────────────────┐
   │ BibliotecaService depende de abstraoes   │
   │ (Material, Usuario) nao implementacoes    │
   │ Facilita testes e extensoes              │
   └──────────────────────────────────────────┘
